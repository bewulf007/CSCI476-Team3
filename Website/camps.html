Camps

Computing and Jewelry Design

The camp will introduce students to computing, digital modeling, and additive manufacturing techniques. The goal is to show middle school aged students that learning about computing can be an enjoyable experience. The students will Rhino速 a computer-aid design (CAD) program to design 3D objects for 3D printing. We will also tour 3D Systems速 where students can learn more about the 3D printing process and industry. This year we will introduce a programming language called Processing. This language is used in electronic arts, new media arts and visual design.

Computing and Robotics

Students will build and program robots from Lego Nxt Robots. Students will be introduced to programming through programming environment called 'Alice'. Developed by Carnegie Mellon University, Alice is a programming environment that supports the creation of 3D animation.

Web design with an introduction to mobile computing

Students will learn how to build a web page and some best practices for a well-designed site. We will look at how to design that site for mobile viewing. Mobile application development will be introduced. (Loaner devices will be provided.) We will also a local web development company to discover how they work.
